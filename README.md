# 💻 DevTinder

DevTinder is a full-stack web application that allows developers to connect with each other — inspired by swipe-based networking platforms.

It enables users to create profiles, send connection requests, and manage developer connections efficiently.

---

## 🚀 Tech Stack

### Frontend
- React (Vite)
- Redux Toolkit
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication

---

## 📂 Project Structure
DevTinder/
│
├── devTinder-web/ # Frontend
└── dev_tinder/ # Backend

---

## 🔥 Features

- 🔐 User Authentication (JWT-based)
- 👤 Profile creation & editing
- 🤝 Send / Accept / Reject connection requests
- 📰 Developer feed system
- 🛡 Protected routes
- 🌐 REST API architecture

---

## 🛠 Installation

### Clone the repository
git clone https://github.com/HimanshuVerma5/devTinder.git

cd devTinder

### Backend Setup

cd dev_tinder
npm install
npm run dev 

### Frontend Setup

cd devTinder-web
npm install
npm run dev
